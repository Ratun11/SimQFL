About Quantum Federated Learning

Quantum Federated Learning (QFL) is a machine learning technique that trains algorithms across multiple decentralized devices without exchanging raw data. This approach helps protect users' privacy and reduces communication costs.

QFL Framework

The QFL framework involves \( N \) distributed clients collaboratively training a shared quantum neural network (QNN) model over \( R \) global communication rounds. In each global round \( r \in \{1, \dots, R\} \), clients \( n \in \{1, \dots, N\} \) perform \( K \) local training epochs on their private datasets \( D_n \) without sharing raw data. After local training, each client sends its updated model parameters to a central server, which aggregates them to update the global model.

Quantum Encoding:
Each client \( n \) at global round \( r \) maps classical input data \( w \in \mathbb{R}^d \) into a quantum state using amplitude encoding. This method efficiently embeds classical vectors into quantum states by encoding their components into the amplitudes of computational basis states. For a \( 2^{Q_n} \)-dimensional input vector, only \( Q_n \) qubits are needed. Given a classical vector \( w \in \mathbb{R}^{2^{Q_n}} \), its amplitude-encoded quantum state is:

$$ |\psi_{\text{enc}}(w)\rangle = \frac{1}{\|w\|} \sum_{i=0}^{2^{Q_n}-1} w_i |i\rangle $$

where \( \|w\| \) is the \( \ell_2 \)-norm of the vector, and \( |i\rangle \) denotes the computational basis states. This encoding can be represented as:

\( |\psi_{\text{enc}}(w)\rangle = U_{\text{enc}}(w) |0\rangle^{\otimes Q_n} \)

Local model training:
Each client applies a PQC \( U(\omega_{n,r}^{k}) \) to the encoded state \( |\psi_{\text{enc}}(w)\rangle \). The PQC consists of \( L \) layers combining trainable single-qubit gates (e.g., \( R_x \), \( R_y \), \( R_z \)) and entangling gates (e.g., CNOT). The unitary is:

$$ U(\omega_{n,r}^{k}) = \prod_{d=1}^L U_d(\omega_{n,r,d}^{k}) $$

where \( U_d(\omega_{n,r,d}^{k}) = \exp\left(-i \frac{\omega_{n,r,d}^{k}}{2} G_d\right) \) and \( G_d \) is a Hermitian generator (e.g., Pauli string). The output state is:

$$ |\psi_{\text{out}}(w, \omega_{n,r}^{k})\rangle = U(\omega_{n,r}^{k}) |\psi_{\text{enc}}(w)\rangle $$

Measurement using observable \( O \) (e.g., Pauli-Z) gives:

$$ f(w, \omega_{n,r}^{k}) = \langle \psi_{\text{out}} | O | \psi_{\text{out}} \rangle $$

With \( M \) measurement shots: \( \hat{f}_{n,r}^{k}(w, \omega_{n,r}^{k}) = \frac{1}{M} \sum_{j=1}^{M} H_j \), where \( H_j \) are measurement results.

For data point \( (w, y) \), loss is \( \ell(y, \hat{f}_{n,r}^{k}(w, \omega_{n,r}^{k})) \). Over a batch \( B_n^k \), batch loss:

$$ L_{n,r}^{k,B}(\omega_{n,r}^{k}) = \frac{1}{|B_n^k|} \sum_{(w, y) \in B_n^k} \ell(y, \hat{f}_{n,r}^{k}(w, \omega_{n,r}^{k})) $$

Gradients use parameter shift:

$$ \frac{\partial f(w, \omega_{n,r}^{k})}{\partial \omega_{n,r,d}^{k}} = \frac{1}{2} \left[f(w, \omega_{n,r}^{k} + \frac{\pi}{2} e_d) - f(w, \omega_{n,r}^{k} - \frac{\pi}{2} e_d)\right] $$

Update rule:

$$ \omega_{n,r}^{k+1} = \omega_{n,r}^{k} - \eta \nabla_{\omega} L_{n,r}^{k,B}(\omega_{n,r}^{k}) $$

Final local update after \( K \) epochs: \( \omega_{n}^{(r)} = \omega_{n,r}^{K} \).

Federated Aggregation:
Server broadcasts \( \boldsymbol{\omega}_{\text{global}}^{(r)} \), clients initialize \( \omega_{n,r}^{0} \) accordingly. After training, clients send \( \omega_n^{(r)} \), and server aggregates:

$$ \boldsymbol{\omega}_{\text{global}}^{(r+1)} = \frac{1}{D} \sum_{n=1}^{N} |D_n| \boldsymbol{\omega}_n^{(r)} $$

where \( D = \sum_{n=1}^{N} |D_n| \). This continues until convergence or round \( R \).

Quantum Federated Learning (QFL) Framework Algorithm

Input: Global rounds \( R \), local epochs \( K \), clients \( \mathcal{N} = \{1, \dots, N\} \), learning rate \( \eta \), number of shots \( M \)

Initialization: Initialize global model parameters \( \boldsymbol{\omega}_{\text{global}}^{(0)} \)

[1] For each global round \( r = 1, \dots, R \):
Server broadcasts \( \boldsymbol{\omega}_{\text{global}}^{(r)} \) to all clients
Each client \( n \in \mathcal{N} \) does:
Initialize local parameters \( \omega_{n,r}^{0} = \boldsymbol{\omega}_{\text{global}}^{(r)} \)
For each local epoch \( k = 0, \dots, K{-}1 \):
Sample mini-batch \( B_n^k \subseteq D_n \)
For each data point \( (w, y) \in B_n^k \):
Encode input using amplitude encoding (Eq. 1)
Apply PQC to get output state (Eq. 3)
Compute measurement expectation (Eq. 4)
Evaluate sample loss \( \ell(y, \hat{f}) \)
Compute gradients using parameter shift rule (Eq. 5)
Update model using gradient descent (Eq. 6)
Send updated parameters \( \omega_n^{(r)} \) to server
Server aggregates updates using weighted average (Eq. 7)

Output: Final global model \( \boldsymbol{\omega}^* = \boldsymbol{\omega}_{\text{global}}^{(R)} \)
